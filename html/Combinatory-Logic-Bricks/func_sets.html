text ‚ÄπFunctions feature a monoidal structure. The identity function is a nullary operation 
 (i.e. a "constant"). It corresponds to the ‚Äπ‚ùôI‚Ä∫ combinator. Function composition is the main binary
  operation between functions and corresponds to the ‚Äπ‚ùôB‚Ä∫ combinator.‚Ä∫

lemma "‚ùôI ‚àò f = f" unfolding comb_defs ..                   ‚Äï ‚Äπidentity 1‚Ä∫
lemma "f ‚àò ‚ùôI = f" unfolding comb_defs ..                   ‚Äï ‚Äπidentity 2‚Ä∫
